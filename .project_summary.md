## What Is MBFD Hub?

MBFD Hub is the Miami Beach Fire Department's internal operations platform — a fully containerized web application running on a dedicated VPS at `145.223.73.170`. It provides:
- **Daily operational forms** (apparatus checkout, inspections, defect reporting)
- **Station inventory management** (PIN-gated, threshold alerts, supply requests)
- **Capital project tracking** (over and under $25k)
- **Workgroup/evaluation management** (Eval Feedback Hub)
- **Training resource management**
- **Personnel, apparatus, and equipment administration**

## Key Features Implemented

### Station Inventory V2 (Complete)
- PIN-gated station access (default PIN: 1234)
- 35 inventory items across 5 categories
- 50% PAR threshold alerts (green/yellow/red)
- Special Supply Request workflow
- Full audit trail (employee name + shift)
- Admin dashboard with low-stock badges

### Workgroup / Eval Feedback Hub (Complete — 2026-02-28)
- Third Filament panel at `/workgroups`
- Workgroup creation and management
- Evaluation template builder
- Candidate product management
- Session-based evaluation flows
- Scoring, ranking, and finalists
- Specialized exporters (scores, finalists, feedback, completion status)

### Pump Simulator (Complete — 2026-03-03)
- Public React SPA at `/pump-simulator`
- ROAD/PUMP mode toggle
- Engine RPM, pressure gauges, valve controls
- Real-time physics, cavitation detection

### CSV / XLSX Export Feature (Complete — 2026-03-03)
- Package: `pxlrbt/filament-excel ^2.5`
- **All 43 Filament resource/relation manager tables** now have:
  - Header **Export** button (full table, respects active filters)
  - Bulk **Export Selected** action (checked rows only)
  - Both `.xlsx` and `.csv` format choices
- Coverage: 15 Logistics resources + 12 Logistics Relation Managers + 3 Training resources + 9 Workgroup resources + 4 Workgroup Relation Managers
