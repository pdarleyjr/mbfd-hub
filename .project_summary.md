## What Is MBFD Hub?

MBFD Hub is the Miami Beach Fire Department's internal operations platform — a fully containerized web application running on a dedicated VPS at `145.223.73.170`. It provides:
- **Daily operational forms** (apparatus checkout, inspections, defect reporting)
- **Station inventory management** (PIN-gated, threshold alerts, supply requests)
- **Capital project tracking** (over and under $25k)
- **Workgroup/evaluation management** (Eval Feedback Hub)
- **Training resource management**
- **Personnel, apparatus, and equipment administration**
- **Google Sheets auto-sync** (Fire Apparatus → Equipment Maintenance tab)

## Key Features Implemented

### Google Sheets Apparatus Sync (Complete — 2026-03-03)
- One-way auto-sync: Fire Apparatus admin page → `Equipment Maintenance` Google Sheet tab
- Target spreadsheet: `1u9MYILAkfEaMfNZnBujvB1J0J33Ha8TybWCd_mVMJC4` (sheetId: `1714038258`)
- Column mapping: A=Designation, B=Vehicle#, C=Status, D=Location, E=Comments, F=Reported
- Trigger: Eloquent observer fires on any apparatus save; stamps `reported_at`; dispatches `SyncApparatusToSheetJob`
- Manual recovery: `artisan apparatus:sync-sheet [--dry-run] [--force]`
- Admin UI: "Sync to Google Sheet" header action (feature-flagged)
- Credentials: service account JSON stored at `/root/secrets/` (outside repo)

## ⚠️ Monitoring Notes

- **High-volume export risk**: `InspectionResource` and `StationInventorySubmissions` could exceed ~5,000 rows over time → migrate those to Filament's native queue-backed exporter if PHP memory issues arise during exports
- **NocoBase Phase 6**: Blocked on CE limitation (http-api datasource requires Pro). Workaround: `plugin-workflow-request` to call Baserow REST API internally
- **Redis Queue**: Database-backed queue via `QUEUE_CONNECTION=database` — monitor for job failures in Laravel Horizon or `artisan queue:work`
- **Google Sheets Sync**: If sync jobs fail repeatedly, check `/root/secrets/google_service_account.json` permissions and quota limits. Use `artisan apparatus:sync-sheet --force` for manual recovery.
